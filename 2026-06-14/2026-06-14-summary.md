### 1. Cosmos 3: Omnimodal World Models for Physical AI

- arxiv ID: 2606.02800
- 方向：多模态、智能体
- PDF: 2606.02800.pdf

**应用背景：** Physical AI（具身智能）需要统一处理语言、图像、视频、音频和动作序列的世界模型，现有方案通常将视觉语言模型、视频生成器、世界仿真器和动作模型分开构建，难以形成统一框架。

**应用方法：** 提出 Cosmos 3，基于 mixture-of-transformers 架构的全模态世界模型族，支持灵活的输入输出配置，将视觉语言模型、视频生成、世界仿真和动作模型统一到单一框架中。代码、模型权重、合成数据集及评估基准均在 Linux Foundation OpenMDW-1.1 License 下开源。

**效果：** 在多种理解和生成任务上达到新 SOTA。Artificial Analysis 评为最佳开源 Text-to-Image 和 Image-to-Video 模型，RoboArena 评为最佳策略模型。

### 2. Self-Harness: Harnesses That Improve Themselves

- arxiv ID: 2606.09498
- 方向：智能体
- PDF: 2606.09498.pdf

**应用背景：** LLM 智能体的性能由基座模型和 harness（交互框架）共同决定，但 harness 设计高度依赖人工专家，难以随模型多样性扩展。

**应用方法：** 提出 Self-Harness 范式，让 LLM 智能体自主改进自身 harness。流程分三阶段迭代：Weakness Mining（从执行轨迹中识别模型特定失败模式）、Harness Proposal（生成多样化且最小的 harness 修改方案）、Proposal Validation（回归测试后才接受修改）。在 Terminal-Bench-2.0 上使用 MiniMax M2.5、Qwen3.5-35B-A3B、GLM-5 三个模型验证。

**效果：** 三个模型的 held-out pass rate 分别从 40.5% 提升至 61.9%、23.8% 提升至 38.1%、42.9% 提升至 57.1%。定性分析表明 Self-Harness 并非添加通用指令，而是将模型特定弱点转化为具体可执行的 harness 变更。

### 3. EvoArena: Tracking Memory Evolution for Robust LLM Agents

- arxiv ID: 2606.13681
- 方向：智能体、评估基准、RAG与记忆
- PDF: 2606.13681.pdf

**应用背景：** 现有 LLM 智能体评估大多假设静态环境，但真实部署中环境持续变化，智能体需要不断更新知识、技能和行为以适应动态条件。

**应用方法：** 提出 EvoArena 基准套件，将环境变化建模为终端、软件、社交领域的渐进更新序列。同时提出 EvoMem，一种基于补丁的记忆范式，将记忆演化记录为结构化更新历史，使智能体能通过记忆变化推理环境演化。

**效果：** 当前智能体在 EvoArena 上平均准确率仅 39.6%。EvoMem 在 EvoArena 上平均提升 1.5%，在 GAIA 和 LoCoMo 标准基准上分别提升 6.1% 和 4.8%。链式准确率提升 3.7%。机制分析显示 EvoMem 改善了记忆中的证据捕获能力。

### 4. MiniMax Sparse Attention (MSA)

- arxiv ID: 2606.13392
- 方向：高效推理
- PDF: 2606.13392.pdf

**应用背景：** 前沿 LLM 对超长上下文能力需求日益增长（智能体工作流、代码仓库推理、持久记忆等需要处理百万级 token），但 softmax attention 的二次复杂度在部署场景不可持续。

**应用方法：** 提出 MiniMax Sparse Attention (MSA)，基于 Grouped Query Attention (GQA) 构建的分块稀疏注意力。Index Branch 对 KV 块打分并为每个 GQA 组独立选择 Top-k 子集，Main Branch 仅在选中的块上执行精确块稀疏注意力。配合免 exp 的 Top-k 选择和 KV-outer 稀疏注意力 GPU 执行路径，提升 tensor-core 利用率。

**效果：** 在 109B 参数多模态模型上，MSA 性能与 GQA 持平，1M 上下文时每 token 注意力计算量降低 28.4 倍。在 H800 上实现 14.2 倍 prefill 和 7.6 倍 decoding 的墙钟加速。已集成到公开部署的 MiniMax-M3 模型中。

### 5. From AGI to ASI

- arxiv ID: 2606.12683
- 方向：可信安全
- PDF: 2606.12683.pdf

**应用背景：** 构建人类水平的通用人工智能（AGI）已成为各大 AI 组织的近期目标，但 AGI 之后的发展路径（从 AGI 到超级通用智能 ASI）缺乏系统分析框架。

**应用方法：** 建立分析框架研究 AI 在 post-AGI 世界的发展连续体。定义 Universal AI 为理论终点，重点分析从 AGI 到 ASI 的四条可能路径：规模化 AGI、AI 范式转换、递归自我改进、大规模多智能体集体涌现。讨论各路径上的潜在摩擦和瓶颈。

**效果：** 报告指出由于 ASI 进展的高度不确定性，AI 加速发展不能被排除。单一变革性事件（AGI 引入社会）的图景可能不准确，更可能是由 AI 驱动的多领域科技进步引发的一系列变革性社会变化。呼吁全球范围的跨学科研究准备。

### 6. ICA Lens: Interpreting Language Models Without Training Another Dictionary

- arxiv ID: 2606.11722
- 方向：可信安全
- PDF: 2606.11722.pdf

**应用背景：** 在 LLM 表示中寻找可解释方向是理解和控制模型行为的关键。稀疏自编码器（SAE）是主流工具，但训练、存储和评估大规模过完备字典成本高昂，限制了快速探索。

**应用方法：** 重新审视独立成分分析（ICA）作为 LLM 可解释性工具。提出 ICALens，首个实用 ICA 工作流，结合优化的 GPU 并行 FastICA 管线、LLM 特定的稳定性方案和拟合诊断工具，无需逐层训练字典即可进行层分析。在 GPT-2 Small、Gemma 2 2B、Qwen 3.5 2B Base 上验证。

**效果：** ICALens 高效恢复紧凑且人类可解释的方向。在 SAEBench 上，ICA 在稀疏探测中与公开 SAE 相当，在小到中等预算下的定向探测扰动中优于 SAE。结论是 ICA 不应被视为弱基线，而是探索 LLM 表示的高效互补第一视角。

### 7. Bebop: Breaking Entropy Bounds — Accelerating RL Training via MTP with Rejection Sampling

- arxiv ID: 2606.12370
- 方向：高效推理、推理能力增强
- PDF: 2606.12370.pdf

**应用背景：** 强化学习（RL）已成为现代 LLM 的关键组件，但 rollout 阶段仍是 RL 训练管线的主要瓶颈。多 Token 预测（MTP）通过投机解码加速 rollout，但 RL 训练期间 MTP 接受率显著下降，加速效果有限。

**应用方法：** 提出 Bebop，系统性研究 MTP 在 LLM 后训练中的应用。发现 MTP 接受率受模型熵波动约束（与 RL 阶段熵增呈负线性关系）。提出概率拒绝采样缓解熵干扰，以及端到端 TV loss 直接优化多步拒绝采样接受率。采用 pre-RL MTP 训练策略，避免昂贵的在线 MTP 更新。在 Qwen3.5、Qwen3.6、Qwen3.7 模型上验证。

**效果：** 接受率提升至约 95%（约 10% 提升），跨数学推理、代码生成和智能体任务实现额外 25% 推理吞吐增益。异步 RL 训练实现最高 1.8 倍端到端加速。

### 8. InterleaveThinker: Reinforcing Agentic Interleaved Generation

- arxiv ID: 2606.13679
- 方向：智能体、多模态、推理能力增强
- PDF: 2606.13679.pdf

**应用背景：** 当前图像生成器在单图任务上具有出色的写实度和指令遵循能力，但无法实现交错生成（文本-图像序列），而交错生成对视觉叙事、引导和具身操作至关重要。

**应用方法：** 提出 InterleaveThinker，首个使现有图像生成器具备交错生成能力的多智能体管线。Planner 智能体组织输入序列，Critic 智能体评估输出并修正偏差。构建 Interleave-Planner-SFT-80k 和 Interleave-Critic-SFT-112k 数据集用于格式冷启动，开发 Interleave-Critic-RL-13k 通过 GRPO 强化逐步指令修正。采用准确性和逐步奖励实现单步 RL 引导轨迹。

**效果：** 在多个基准上匹配 Nano Banana 和 GPT-5。显著增强基础模型的推理能力，在 4 步 FLUX.2-klein 上 WISE 和 RISE 基准均有大幅提升。

### 9. MaxProof: Scaling Mathematical Proof with Generative-Verifier RL

- arxiv ID: 2606.13473
- 方向：推理能力增强、垂直领域
- PDF: 2606.13473.pdf

**应用背景：** 竞赛级数学证明需要高度可靠的证明生成和验证能力，现有 LLM 在生成竞赛级数学证明方面仍有显著差距。

**应用方法：** 提出 MaxProof，面向 MiniMax-M3 系列的群体级测试时扩展框架。M3 先训练三种证明能力：证明生成、证明验证、基于批评的证明修复，使用纵深防御生成验证器确保低误判率。测试时将模型同时作为生成器、验证器、精炼器和排名器，在候选证明群体中通过锦标赛选择返回最终证明。

**效果：** M3 模型在 IMO 2025 上达到 35/42，USAMO 2026 上达到 36/42，均超过人类金牌线。

### 10. EurekAgent: Agent Environment Engineering is All You Need For Autonomous Scientific Discovery

- arxiv ID: 2606.13662
- 方向：智能体
- PDF: 2606.13662.pdf

**应用背景：** LLM 智能体在自动化科学发现方面展现出日益增长的潜力，但自主科学发现的瓶颈正从规定智能体工作流转向设计智能体环境。

**应用方法：** 提出环境工程范式，从四个维度设计智能体环境：权限工程（有界执行与隔离评估）、工件工程（文件系统与 Git 协作）、预算工程（预算感知探索）、人在环路工程（便于监督和介入）。在数学、核工程、机器学习任务上验证。

**效果：** 在多个数学、核工程和机器学习任务上达到新 SOTA，包括以不到 $11 的 API 成本发现新的 26 圆最优填充结果。代码和结果全部开源。
