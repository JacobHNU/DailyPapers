### 1. Autodata: An Agentic Data Scientist to Create High Quality Synthetic Data

- **作者**: Ilia Kulikov, Chenxi Whitehouse, Tianhao Wu, Yixin Nie (共同一作), Swarnadeep Saha, Eryk Helenowski, Weizhe Yuan, Olga Golovneva, Jack Lanchantin, Yoram Bachrach, Jakob Foerster, Xian Li, Han Fang, Sainbayar Sukhbaatar, Jason Weston (FAIR at Meta)
- **arXiv**: [2606.25996](https://arxiv.org/abs/2606.25996)
- **热度**: 206 bookmarks, 3,563 views
- **方向**: 智能体, 推理能力增强

**问题**: 现有合成数据生成方法（如 Self-Instruct、CoT Self-Instruct）无法直接控制生成数据的难度和质量。标准 prompting 生成的数据要么对弱模型太简单（CS 任务中 weak-strong gap 仅 0.02），要么太难（法律任务中弱 solver 均分仅 0.159，大量 rollout 得分为 0），导致下游 RL 训练信号低效。

**方法**: 提出 Autodata 框架，让 AI agent 充当"数据科学家"角色，通过迭代式数据创建-分析循环来生成高质量合成数据。具体实现为 Agentic Self-Instruct：主编排 agent 调度四个子 agent（Challenger 生成样本、Weak solver 和 Strong solver 分别作答、Judge 评估质量），以 weak-strong solver 的分数差距为接受标准，迭代优化生成数据的难度和质量。此外，框架支持外层 meta-optimization，用内循环标准优化 agent 本身的数据科学能力。

**关键结果**: CS 研究任务中，Agentic loop 平均需 6.59 轮迭代生成一条被接受样本；weak-strong gap 从 CoT 的 0.019 提升至 0.314。用 1.3k 样本 GRPO 训练 Qwen3.5-4B，mean@3 从 CoT 数据的 0.500 提升至 0.632 (+26.4%)。法律推理任务中，Qwen3.5-4B 在 PRBench-Legal 上得分 0.441，超越未微调的 Qwen3.5-397B (0.404)。科学推理任务中，Agentic Self-Instruct 数据训练 avg@8 提升 +3.20%，推理截断率从 23.75% 降至 4.09%。

**主要贡献**: 首次将数据创建形式化为智能体驱动的迭代式数据科学流程；提出可元优化的数据科学家框架；在法律推理中展示 4B 模型通过 agentic 数据训练可超越 397B 基线模型，证明数据质量对模型能力的放大效应。

**局限性**: 实验仅覆盖三个领域，未验证在代码生成、通用对话等更广泛任务上的泛化性；Agentic loop 计算开销显著（CS 任务平均 6.59 轮/样本，法律 4.98 轮/样本）；弱/强 solver 配对选择高度特定，未充分讨论敏感性。

### 2. DSpark: Confidence-Scheduled Speculative Decoding with Semi-Autoregressive Generation

- **作者**: Xin Cheng, Xingkai Yu, Chenze Shao, Jiashi Li 等 (DeepSeek, 北京大学)
- **来源**: [alphaxiv.org](https://www.alphaxiv.org/) (arXiv ID 待发布)
- **热度**: 155 bookmarks, 2,865 views
- **方向**: 高效推理, 垂直领域

**问题**: 推测解码（Speculative Decoding）通过解耦草稿生成与目标验证来加速 LLM 推理，但在高并发实际部署场景中，如何在保证生成质量的同时最大化吞吐量提升仍是核心挑战。

**方法**: 提出 DSpark 框架，采用置信度调度（Confidence-Scheduled）机制与半自回归（Semi-Autoregressive）生成策略相结合的推测解码方案。核心思路是将草稿生成与最终验证分离，利用并行化草稿生成配合灵活的、负载感知的验证机制来提升推理吞吐量，同时通过置信度调度确保生成质量。

**关键结果**: 据报告，DSpark 在真实高并发推理服务中可将大模型推理速度提升 60%-85%，吞吐量提升最高达 400%。

**主要贡献**: 提出面向实际部署的推测解码优化框架；将置信度调度引入推测解码流程以平衡速度与质量。

**局限性**: 论文 arXiv ID 待发布，无法获取完整 PDF 进行详细分析；具体实验设置和基线对比信息有限。

### 3. Orca: The World is in Your Mind

- **作者**: Yihao Wang, Yuheng Ji (Project Lead), Mingyu Cao, Yanqing Shen 等 (OrcaTeam, BAAI). Research Leads: Jiaming Li, Yonghua Lin, Tiejun Huang, Zhongyuan Wang, Pengwei Wang
- **arXiv**: [2606.30534](https://arxiv.org/abs/2606.30534)
- **热度**: 162 bookmarks, 1,506 views
- **方向**: 多模态, 推理能力增强, 智能体

**问题**: 当前主流模型分别优化 Next-Token-Prediction（LLM）、Next-Frame-Prediction（视频生成）或 Next-Action-Prediction（具身控制），各自独立且缺乏对世界状态的统一建模。本文旨在构建一个通用的"世界基础模型"，通过 Next-State-Prediction 范式学习统一的世界隐状态空间，使其能同时支撑理解、预测和行动三类下游任务。

**方法**: Orca 采用 Encoder-Decoder 架构，以预训练 VLM（Qwen3.5）为骨干，从视觉和语言两种模态信号学习世界隐空间。核心设计为两个互补的学习范式：（1）无意识学习——从连续视频中仅通过观测预测下一帧隐状态，捕获密集的自然状态转移；（2）有意识学习——以语言描述的事件作为显式条件，引导跨事件的稀疏语义状态转移，并结合 VQA 生成提供常识约束。预训练数据包含 125K 小时视频和 160M 事件标注。下游阶段冻结骨干，仅训练轻量级模态特定读出头。

**关键结果**: 文本生成（4 个 OOD 基准 zero-shot）：Orca-4B 均分 51.8，超越 Qwen3.5-4B（46.7）及 Emu3.5-34B（29.8）。图像预测（PRICE-V0.1）：Orca-4B 均分 59.8，超越 FLUX.2[klein]（56.1）。动作生成（双臂轮式机器人 5 任务 OOD）：Orca Rule-based 得分 32.4，超越 pi_0.5（29.4）。三个下游读出头性能均随预训练数据增加而单调提升。训练效率：吞吐量从 0.66 提升至 2.91 Samples/Sec/GPU，相比 StarVLA 加速约 4.4 倍。

**主要贡献**: 首个通用世界基础模型的初步实例化，以 Next-State-Prediction 替代传统的 token/frame/action 预测；设计无意识学习与有意识学习两种互补范式；构建 125K 小时视频 + 160M 事件标注的大规模预训练数据集；证实所学隐空间对文本、图像、动作三类下游均有效。

**局限性**: 当前仅使用视觉和语言模态，未纳入音频、触觉、力反馈等物理信号；实验仅在 0.8B 和 4B 规模进行，且仅使用了十分之一的库存数据；4B 模型在三类读出头间已出现 trade-off；具身任务难度有限。

### 4. Scaling the Horizon, Not the Parameters: Reaching Trillion-Parameter Performance with a 35B Agent (Agents-A1)

- **作者**: Agents-A1 Team (上海人工智能实验室). 项目负责人：Bo Zhang, Lei Bai. 科学指导：Wenlong Zhang, Fenghua Ling, Dahua Lin, Bowen Zhou 等
- **arXiv**: [2606.30616](https://arxiv.org/abs/2606.30616)
- **热度**: 51 bookmarks, 794 views
- **方向**: 智能体, 高效推理, 推理能力增强

**问题**: 当前提升长 horizon 智能体能力的路径主要有两条：一是扩大模型参数规模（如 1T 参数模型），但成本高昂且难以复现；二是扩展 agent 的交互 horizon，但缺乏支撑长 horizon 轨迹训练的知识-动作基础设施，且难以将跨领域的异构能力有效整合到统一模型中。

**方法**: 提出 Agents-A1，一个 35B MoE 智能体模型，采用三阶段训练方案：(1) 全域监督微调（Full-domain SFT），在搜索、科学推理、工程、指令遵循等多域长 horizon 数据上对齐基础模型；(2) 分域教师模型训练；(3) 域路由在策略蒸馏（Domain-Routed OPD）+ 显著词汇对齐（SVA），在教师分布的 top-k 显著词汇上进行截断 reverse KL 对齐，通过域归一化防止高频域主导梯度更新。底层构建知识-动作图（KAG），通过 proposer-solver-verifier 自博弈循环扩展和验证，生成平均长度为 45K tokens 的智能体轨迹。

**关键结果**: 在多个长 horizon agent 基准上达到或超越 1T 参数模型：SEAL-0: 56.4（vs Kimi-K2.6 50.5, DeepSeek-V4-pro 55.0）；IFBench: 80.6（vs Kimi-K2.6 73.5）；HiPhO: 46.4（vs DeepSeek-V4-pro 38.7）；FrontierScience-Olympiad: 79.0（vs DeepSeek-V4-pro 78.0）；MolBench-Bind: 56.8（vs Kimi-K2.6 48.7）。

**主要贡献**: 证明通过扩展 agent horizon 而非参数规模，35B 模型可在长 horizon 交互任务中匹配甚至超越 1T 参数模型；构建了长 horizon 知识-动作基础设施（KAG）；提出域路由 OPD + SVA 方法解决跨域推理模式冲突。

**局限性**: 基础模型初始化依赖 Qwen3.5-35B-A3B，能力上限受限于基座模型；长 horizon 场景覆盖仍以特定领域为主，通用性有待验证；"推理前规划"、"行动前反思"等基础原子能力仍有提升空间。

### 5. PhysisForcing: Physics Reinforced World Simulator for Robotic Manipulation

- **作者**: Peiwen Zhang, Yufan Deng, Shangkun Sun, Juncheng Ma, Duomin Wang, Jonas Du, Zilin Pan, Ye Huang, Hao Liang, Songyan Huang, Ruihua Zhang, Enze Xie, Ming-Yu Liu, Daquan Zhou（北京大学 & NVIDIA）
- **arXiv**: [2606.28128](https://arxiv.org/abs/2606.28128)
- **热度**: 50 bookmarks, 663 views
- **方向**: 垂直领域, 智能体, 多模态

**问题**: 当前视频生成模型在机器人操作场景中仍会产生物理上不合理的视频，表现为不连续的运动轨迹、物体穿透、反重力运动、以及机器人-物体交互关系不一致，严重限制了视频模型作为可靠世界模拟器的能力。

**方法**: 提出 PhysisForcing，一种面向物理信息区域的层次化物理对齐训练框架。首先通过点跟踪（CoTracker3）和深度感知前景权重提取物理信息区域掩码；然后在 DiT 中间层特征上施加两个互补损失：（1）像素级轨迹对齐损失——用参考点轨迹监督 DiT 特征的逐点运动，强制局部轨迹连续性；（2）语义级关系对齐损失——将 DiT 特征中物理信息 token 间的成对相似度矩阵与冻结视频理解编码器（VideoDINO）提取的关系结构对齐。所有辅助模型仅在训练时使用，推理时零额外开销。

**关键结果**: R-Bench：PF-Cosmos 达到 63.8 分（基线 +9.2%），PF-Wan 达到 62.0 分（基线 +22.3%），均超越最强商业模型 Wan2.6（60.7）。PAI-Bench 机器人域：PF-Cosmos 达到 85.2。下游策略学习（RoboTwin 2.0）：平均成功率从 68.2% 提升至 72.8%。WorldArena 闭环动作规划：成功率从 16.0% 提升至 24.0%，超越所有世界模型规划器。

**主要贡献**: 首次将机器人视频生成中的物理合理性形式化为层次化、区域聚焦的对齐问题；提出像素级轨迹对齐和语义级关系对齐的双损失框架，不增加推理开销；验证了物理对齐的视频模型可作为更强表征服务于下游策略学习和动作规划。

**局限性**: 依赖外部点跟踪器提供参考轨迹，跟踪误差可能传导至训练监督信号；部分任务出现性能回退（如 shake_bottle: 97.5%→94.5%），说明物理对齐对已高度优化的简单动作可能引入过约束；仅在 DiT 架构上验证。

### 6. MOPD: Multi-Teacher On-Policy Distillation for Capability Integration in LLM Post-Training

- **作者**: Wenhan Ma, Jianyu Wei, Liang Zhao, Hailin Zhang, Bangjun Xiao, Lei Li, Qibin Yang, Bofei Gao, Yudong Wang, Rang Li, Jinhao Dong, Zhifang Sui, Fuli Luo
- **arXiv**: [2606.30406](https://arxiv.org/abs/2606.30406)
- **热度**: 35 bookmarks, 484 views
- **方向**: 推理能力增强, 高效推理

**问题**: 现代 LLM 后训练阶段依赖强化学习提升特定能力，但不同领域需要不同的 RL 训练流程。核心难题在于：如何将多个领域专家模型的能力高效整合到一个统一模型中，同时避免跨域干扰、能力遗忘、曝光偏差和权重融合不稳定？

**方法**: 提出多教师在线策略蒸馏（MOPD）三阶段后训练范式。Stage 1 通用 SFT；Stage 2 从 SFT checkpoint 出发并行训练各领域 RL 专家（教师）；Stage 3 以学生模型自身 rollout 采样轨迹，由对应领域教师提供 token 级 log-probability 监督，通过最小化 token 级逆 KL 散度进行策略空间蒸馏。提供策略梯度和 Top-k 蒸馏两种等价实现。工程上将教师部署为独立 prefill 服务，与 RL 采样异步执行。

**关键结果**: 在 Qwen3-30B-A3B 上，MOPD 归一化得分 0.937，比最强基线 Mix-RL（0.882）高出 5.5 分，三个领域分数范围 [0.91, 0.95]，波动仅 0.044。样本效率显著优于 Mix-RL：MOPD 在 IF 领域约 25K 样本、SWE 领域约 30K 样本即达到教师水平，而 Mix-RL 需要 150-180K 样本。在工业级 MiMo-V2-Flash（309B 参数，五领域）上验证全面有效。多轮迭代演化：Iter-2 MOPD 从 0.937 提升至 0.986。

**主要贡献**: 首次在策略空间通过 token 级密集监督实现多教师能力整合，同时满足密集优化、on-policy 训练和教师并行化三项关键属性；在工业级 309B 模型上完成部署验证；深入分析同源教师对蒸馏稳定性的决定性作用。

**局限性**: 高度依赖教师与学生共享相同初始化的"同源"特性，引入更强但异构的外部教师会导致训练崩溃；三阶段流程工程部署门槛较高；主要实验仅覆盖 Math、IF、SWE 三个领域。

### 7. DOPD: Dual On-policy Distillation

- **作者**: Xinlei Yu, Gen Li, Qingyi Si, Guibin Zhang, Yuqi Xu, Congcong Wang, Shuai Dong, Kaiwen Tuo, Xiangyu Zeng, Kaituo Feng, Qunzhong Wang, Yang Shi, Xiaobin Hu, Xiangyu Yue, Jiaqi Wang, Shuicheng Yan (NUS, MMLab CUHK, PKU, Explore Academy JD)
- **arXiv**: [2606.30626](https://arxiv.org/abs/2606.30626)
- **热度**: 35 bookmarks, 416 views
- **方向**: 高效推理, 推理能力增强

**问题**: 在线策略蒸馏（OPD）在引入特权信息（如经验证的推理提示、结构化视觉标注）以提升监督质量时，存在"特权幻觉"（privilege illusion）失效模式——教师与学生之间的表观性能差距混淆了可迁移的能力差距与不可学习的信息不对称差距，学生模型会拟合特权信息带来的捷径，导致训练后期性能退化与熵坍缩。

**方法**: 提出 DOPD（Dual On-policy Distillation），一种优势感知双重蒸馏范式。核心思路是：当教师和学生均获得特权信息时，计算每个 token 的"特权优势差距"A，结合预测概率将 token 动态路由至四种蒸馏策略：(1) 低 A + 高概率 → 轻量教师蒸馏；(2) 低 A + 低概率 → 弱自正则化；(3) 高 A + 教师高概率 → 强全词汇表教师蒸馏（JS 散度）；(4) 高 A + 学生高概率 → 轻量特权学生蒸馏。

**关键结果**: LLM 场景（Qwen3-8B → Qwen3-1.7B）：DOPD 在 8 个基准上平均得分 51.4，较 Vanilla OPD（43.9）提升 7.5 分，恢复教师-学生原始差距的 89.8%。在 AIME25（23.3 vs 20.2）、LCBv5（27.1 vs 23.6）上超越教师策略。VLM 场景（Qwen3-VL-8B → Qwen3-VL-2B）：DOPD 平均 58.4，较 Vanilla OPD（52.4）提升 6.0 分。五组不同规模模型对上均取得 6.2-10.6 分一致提升。

**主要贡献**: 首次系统识别并定义 OPD 中"特权幻觉"失效模式；提出特权优势差距作为 token 级信号质量度量；设计四路动态路由的双重蒸馏机制，在 LLM 和 VLM 双场景下均显著优于 9 种基线方法。

**局限性**: 特权信息生成依赖 GPT-5.4，引入额外数据准备成本；方法涉及较多超参数，敏感性未充分讨论；仅在 Qwen3 系列上验证，跨架构泛化性有待确认。

### 8. ScaleAware-JEPA: Latent Representation for Discovery in Multiscale Physical Fields

- **作者**: Guang-Xing Li
- **arXiv**: [2606.29723](https://arxiv.org/abs/2606.29723)
- **热度**: 30 bookmarks, 379 views
- **方向**: 垂直领域

**问题**: 连续物理场（如湍流密度、分子气体分布、城市夜光等）具有多尺度结构，但现有自监督学习方法在固定图像坐标系下定义上下文与预测目标，使用固定尺寸的 patch 进行遮蔽，与物理场跨连续尺度的组织方式不匹配。核心挑战是：如何在不需要标签或预定义分割规则的前提下，为复杂多尺度物理场构建稠密的、可回溯的隐空间坐标。

**方法**: ScaleAware-JEPA 通过约束扩散分解（CDD）将输入标量场演化为像素对齐的多尺度分量（扩散时间 t 对应特征尺度 λ ∝ √t）。每个尺度分量由独立的 scale-aware adapter 处理后，经粗到细的残差融合送入 dense ConvNeXt V2 编码器生成全分辨率隐空间映射。遮蔽框大小与对应分量的扩散尺度绑定，使预测任务在细、中、粗各尺度上分别进行。训练采用 JEPA 范式，在线分支预测 EMA 目标分支的隐表示。

**关键结果**: 在 MHD 湍流密度场上，金字塔遮蔽在 1.2×σ_s 时目标有效秩和铰链比均处于上升段，提供了比固定框遮蔽更可控的上下文-预测关系。隐空间回映在三个物理机制迥异的系统中均产生空间连贯的形态分区：MHD 场中自监督区分了与 Alfvén Mach 数对应的磁控区域和动能主导区域；成都夜光场中分离出紧凑城区、光走廊、郊区结构和低强度背景；NGC 3627 分子气体场中捕获旋臂与臂间的大尺度对比。

**主要贡献**: 首次将 JEPA 的隐空间预测与物理场的固有尺度层级绑定；提出尺度感知金字塔遮蔽，遮蔽框大小由扩散尺度决定；在三个物理机制迥异的系统上验证了跨域适用性，无需标签、目录或分割规则。

**局限性**: 局部 ConvNeXt 感受野有限，对大尺度连续结构只能恢复局部形态特征；仅针对二维标量场验证，未扩展到矢量场或三维体积数据；评估以定性可视化为主，缺少定量对比基准；单一作者工作，大规模高分辨率场的可扩展性未充分讨论。

### 9. AdaJEPA: An Adaptive Latent World Model

- **作者**: Ying Wang, Oumayma Bounou, Yann LeCun, Mengye Ren
- **arXiv**: [2606.32026](https://arxiv.org/abs/2606.32026)
- **热度**: 14 bookmarks, 203 views
- **方向**: 高效推理, 推理能力增强, 智能体

**问题**: 现有潜在世界模型（如 JEPA）在推理阶段通常将模型参数冻结，导致在测试环境出现分布偏移时——包括视觉变化、物理动力学变化、未见过的几何形状或迷宫布局——预测误差累积，规划失败。核心挑战在于如何让世界模型在部署时持续自适应校准。

**方法**: 提出 AdaJEPA，在模型预测控制（MPC）闭环中执行测试时自适应。核心为"规划-执行-自适应-重规划"循环：(1) 用当前模型规划动作序列；(2) 执行第一个动作 chunk 并观测下一状态；(3) 利用观测到的转移作为自监督信号，对编码器末层和预测器末层进行梯度更新（仅需 1 步梯度下降，学习率 η_pred=5×10⁻⁴, η_enc=10⁻⁵，维护最近 5 个样本的 replay buffer）；(4) 用更新后的模型重规划。不依赖额外专家标注或奖励标签。

**关键结果**: 形状偏移：在未见过的 PushObj 形状上，规划成功率近乎翻倍。低数据场景：在 K=1, N=1k 条件下，成功率从 28.1% 提升至 60.8%，超越了使用 16 倍数据量训练的冻结模型（43.5%）。迷宫布局偏移：GD 成功率从 53.3% 提升至 78.7%（+25.3%）。延迟开销极小：每次 MPC 重规划步骤仅增加 0.01-0.03 秒（单张 H200）。在 3 种不同 JEPA 变体上均有一致提升。

**主要贡献**: 首次在 JEPA 世界模型的 MPC 规划闭环中进行测试时自适应，将学习和规划紧密耦合；方法极其轻量——仅需 1 步梯度更新、仅更新末层，延迟增加可忽略；在四类分布偏移上系统验证有效性；方法与底层世界模型解耦，在多种 JEPA 变体上验证通用性。

**局限性**: 自适应性受限于预训练表征的覆盖范围，对训练中完全缺失的特征提升有限；最佳适配目标与环境相关，尚无统一的自动选择策略；仅在 PushT 和 PointMaze 两个中等复杂度基准上验证。

### 10. Diagnosing and Mitigating Context Rot in Long-horizon Search

- **作者**: Shijie Xia（上海交通大学）, Yikun Wang（复旦大学）, Zhen Huang（复旦大学）, Pengfei Liu（上海交通大学）
- **arXiv**: [2606.29718](https://arxiv.org/abs/2606.29718)
- **热度**: 15 bookmarks, 165 views
- **方向**: 智能体, 推理能力增强, RAG与记忆

**问题**: 在长程搜索（Deep Search）场景中，LLM Agent 需要执行数十或上百次搜索工具调用，累积大量上下文。随着上下文不断增长，模型能力退化——即"上下文腐化"（Context Rot）。然而现有研究对真实 Agent 搜索任务中的腐化现象缺乏系统诊断。

**方法**: 构建细粒度终止状态分类体系（放弃、不确定回答、自信回答、无回答），在 4 个旗舰开源模型（Qwen3.5-397B-A17B、GLM-4.7、GLM-5.0、MiniMax-M2.5）和 3 个基准（BrowseComp、BrowseComp-Plus、xbench-DeepSearch）上系统评估。通过上下文剪枝实验揭示因果关系，并评估三类七种上下文管理方法及基于腐化感知的拒绝采样策略。

**关键结果**: 随轨迹长度增加，模型准确率急剧下降，BrowseComp 上 Qwen3.5 的放弃+不确定错误率达 53.4%。上下文窗口大小并非性能瓶颈。完全移除累积上下文可使放弃和不确定错误率接近零，但代价是未完成轨迹大幅增加。混合策略 KeepLatest(w/sum.) 在多数据集平均准确率上最优（Qwen3.5 达 62.9%，vs ReAct 54.4%，+8.5pp）。腐化感知过滤在三种聚合策略上均带来 2.6%-4.9% 提升，Qwen3.5 + Filter 在 MV 聚合下平均准确率达 69.4%。

**主要贡献**: 首次在真实长程 Agent 搜索任务中系统揭示并定义上下文腐化现象；通过上下文剪枝实验证明腐化与累积上下文内容相关（而非仅取决于长度）；对三类七种管理方法提供系统评估和选择指南；提出可组合的腐化感知过滤策略。

**局限性**: 仅覆盖开源模型，排除闭源模型（因加密推理内容），结论在闭源模型上的适用性未知；BrowseComp 仅采样 100 条，样本规模有限；上下文管理触发阈值为人工设定，缺乏自适应机制；仅验证搜索类 Agent，向其他长程任务的推广有待确认。
